# Sudoku solver 
**sudoku-solver** with *image processing* (opencv), *machine learning* (sklearn), *docker-compose* and *web-server* (Flask)

1. Clone
    ```bash
    git clone https://github.com/happy-unicorn/sudoku-solver.git
    ```
2. Go
    ```bash
    cd sudoku-solver
    ```
3. Run
    ```bash
    docker-compose up
    ```
